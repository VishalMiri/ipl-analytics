# 🏏 IPL Analytics Dashboard | Power BI

An interactive **Power BI dashboard** that analyzes Indian Premier League (IPL) data to uncover insights into team performance, player statistics, match outcomes, and tournament trends. The dashboard is designed to help users explore IPL data visually through dynamic charts, KPIs, and filters.

---

## 📌 Project Overview

This project demonstrates how Power BI can be used to transform raw cricket data into meaningful business-style insights through interactive visualizations.

The dashboard enables users to:

* Analyze overall IPL performance
* Compare teams across multiple seasons
* Explore player statistics
* Understand winning patterns and trends
* Interactively filter data by team, season, venue, and other dimensions

---

## 🚀 How to Use

1. Clone this repository.
2. Open **IPL_Analysis.pbix** using Power BI Desktop.
3. If prompted, update the data source path to the **data/** folder.
4. Refresh the data.
5. Explore the dashboard using the slicers and interactive visuals.
